# Guide Étudiant — Terminal Learning

> Version : 1.0 (à compléter — Phase 7c)
> Audience : étudiants inscrits
> Accès in-app : `/help/student`

---

## 1. Premiers pas

- [ ] Créer un compte ou se connecter (GitHub / Google / email)
- [ ] Choisir son environnement : Linux, macOS ou Windows
- [ ] Commencer par le Module 1 — Navigation de base

## 2. Comprendre sa progression

- **Niveau CEFR** : A1 (Découverte) → C2 (Expert)
- **Seuil de maîtrise** : 80% par module pour débloquer le suivant
- **Quiz final** : requis à partir du niveau B1 avant de passer au module suivant
- Les exercices non réussis peuvent être retentés sans limite

## 3. Choisir son track

| Track | Public cible | Modules prioritaires |
|-------|-------------|----------------------|
| Full-Stack Developer | Web, APIs, outils dev | Navigation → Git → GitHub |
| System Administrator | Linux, réseaux, serveurs | Permissions → Réseau → SSH |
| Automation & Scripting | Scripts, CI/CD | Variables → Scripts → Cron |

## 4. Utiliser l'agent IA tuteur

> Fonctionnalité disponible à partir de la Phase 7b.

L'agent IA nécessite votre propre clé API (gratuit pour un faible usage).
→ Le Module 11 "L'IA comme outil dev" vous guide pour l'obtenir.

## 5. FAQ

**Ma progression a disparu après déconnexion**
→ Vérifiez que vous êtes bien connecté. La progression est sauvegardée dans le cloud uniquement pour les comptes connectés.

**Je suis bloqué sur un exercice**
→ Après 2 tentatives, un indice partiel apparaît. Après 4 tentatives, la commande est suggérée.

**Comment changer d'environnement ?**
→ Menu utilisateur (avatar en haut à droite) → switcher d'environnement.

**Je veux réinitialiser ma progression sur un module**
→ Profil → Ma progression → [Module] → Réinitialiser.
