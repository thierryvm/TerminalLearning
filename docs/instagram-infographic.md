# Infographic Mockup — Instagram Strategy

> This is the text mockup for the main strategy infographic.
> Ready to be transformed into a visual design (Figma / Canva).
> Format: Square (1080x1080) or Vertical (1080x1350)

---

## Visual Specifications

| Property          | Value                                              |
| ----------------- | -------------------------------------------------- |
| Background        | Dark gradient (#0a0a0f → #111118)                  |
| Title font        | Monospace (JetBrains Mono, Fira Code, or similar)  |
| Body font         | Sans-serif (Inter, Geist, or similar)              |
| Accent colors     | Module colors from the app (#22c55e, #3b82f6, #a855f7, #f59e0b) |
| Glow              | Subtle, on block titles only — not on body text    |
| Layout            | 4 stacked blocks, single vertical flow             |
| Reading direction | Top → Bottom                                       |
| Max text per line | ~40 characters                                     |
| Decorative        | Minimal — no textures, no complex patterns          |

## Composition Rules (for designer)

1. **4 blocks maximum** — no exceptions
2. **Single reading direction** — top to bottom only
3. **One arrow between each block** — subtle, not decorative
4. **No crossed paths** or branching arrows
5. **Block titles** get subtle glow effect (match module colors)
6. **Body text** stays clean white, no glow
7. **Bullet points** use module accent colors as dot color
8. **Footer** is separated by a thin horizontal line
9. **Tagline** ("Simple · Régulier · Utile") is slightly larger than the secondary line
10. **No granular textures**, no noise overlay, no gradients on text
11. **Readable in under 10 seconds**
12. **Terminal aesthetic preserved** — monospace title, dark bg, glow accents

## What Was Removed (vs. previous infographic)

| Removed                              | Why                                        |
| ------------------------------------ | ------------------------------------------ |
| Duplicate automation blocks          | Redundant — workflow covers it              |
| Multiple dotted arrow paths          | Confusing — one direction is enough         |
| Tiny example text                    | Unreadable on mobile                        |
| Excess body copy                     | Infographic ≠ document                     |
| Plan/validation/publication overlap  | Merged into single workflow block           |
| Decorative elements over content     | Clarity > decoration                        |
